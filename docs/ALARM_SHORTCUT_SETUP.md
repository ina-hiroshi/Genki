# 目覚まし連携の設定ガイド

Genki の「元気だよ」を、毎朝の目覚まし（アラーム）停止と連動させる手順です。
iOS では純正アラームに直接フックできないため、**ショートカットのオートメーション**を使います。

## 仕組み

```
純正アラームを止める
  → オートメーション「アラームが停止したとき」が発火
  → Genki の「元気だよを送る」インテントを実行
  → 家族に「○○が起きました／元気だよ」が届く
```

Genki は App Intent `CheckInIntent`（表示名「元気だよを送る」）と App Shortcut を提供しています。
バックグラウンドで完結するため、アプリを開かずに送信できます。

## 設定手順（ユーザー向け）

1. iPhone の「ショートカット」アプリを開く。
2. 下部の「オートメーション」タブ →「＋」→「個人用オートメーションを作成」。
3. トリガーで「アラーム」を選び、「停止したとき」を選択。対象の目覚ましアラームを指定。
4. 「次へ」→「アクションを追加」→ Genki の「**元気だよを送る**」を選ぶ
   （検索で「元気だよ」または「Genki」）。
5. 「すぐに実行」をオンにして（確認なしで自動実行）、保存。

これで、設定したアラームを止めるたびに自動で「元気だよ」が家族へ送信されます。

## 補足

- Siri からも「〇〇（アプリ名）で元気だよを送る」で実行できます。
- 同じ日に2回目以降は重複送信されません（1日1回）。
- アプリ内（ホーム / チェックインタブ / Apple Watch）からも手動で送れます。

- インテント実装: [`Genki/Intents/CheckInIntent.swift`](../Genki/Intents/CheckInIntent.swift)

  App Group のスナップショットを更新します。
